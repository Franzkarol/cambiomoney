General settings for Every page

  • 1. Change the title
    <title>Roysha - Currency HTML5 Template</title>

  • 2. Change the logo

  • 3. Change google fonts in (style.css)
    @import url("https://fonts.googleapis.com/css?family=Rubik:33,430,500,700&display=swap");

Preloader customization

  • 1. Change preloader images (style.css)

  • 2. Change preloader background (style.css)

  • 3. Available jQuery options for preloader (main.js)

Navigation customization

Navigation's are inside 'nav-header' section. Change navigation with your own text for every page.

  • 1. Change / Add your nav items for primary menu

Owl carousel customization

  • 1. Owl carousel css
    <link rel="stylesheet" href="css/owl.carousel.min.css">

  • 2. Owl carousel js
    <script src="js/owl.carousel.min.js"></script>

  • 3. Customize Owl carousel jQuery options (scripts.js)

Fancybox customization

  • 1. Fancybox css
    <link rel="stylesheet" href="css/jquery.fancybox.min.css">

  • 2. Fancybox js
    <script src="js/jquery.fancybox.min.js"></script>

  • 3. Add fancybox to HTML element

    Note:
    1. add video url in 'href' tag.
    2. add 'data-fancybox' as attribute.

Countdown jQuery customization

  • 1. Countdown js
    <script src="js/jquery.countdown.min.js"></script>

  • 2. Countdown jQuery options (scripts.js)
